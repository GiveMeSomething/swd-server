## Installation

### 1. Install Node.js (LTS)

- https://nodejs.org/en/

### 2. Install yarn

- https://classic.yarnpkg.com/en/docs/install#windows-stable

### 3. Install Docker

- https://www.docker.com/

## Running the app

### 1. Install dependencies with

```
yarn
```

### 2. Start Docker container

```
yarn docker:start
```
